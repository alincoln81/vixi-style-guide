Fonts

  • Inter is the only font that should be used for all of Vixi Suite and it's related products and marketing materials.
    This is a Google Font and it can be downloaded here

Confirmation Dialogs & Alerts

  • Alert & Confirmation Dialogs should NOT have rounded edges.
    • However, the buttons contained in them should still have a 5px border radius
  • Confirmation dialogs should apear as a model in the center of the page with a semi-opaque background covering the whole page
  • Alert dialogs should appear 20px from the top of the page and be horizontially centered.

Tooltips & System Messages

  • Tooltips & System Messages should NOT have rounded edges.
  • System Messages should appear 20px from the bottom of the page and be horizontially centered.
  • Tooltips should appear 5px to above the element on hover, and be cetered, see the tooltips on the Icons for an example.
